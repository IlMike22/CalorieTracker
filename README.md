# CalorieTracker
A small Android app for tracking your calories.

- kotlin
- jetpack compose
- local datasource with room
- remote datasource with retrofit
- dependeny injection with dagger hilt
- coroutines
- mvvm with use cases
- multi module project
- channel and flow for ui events
